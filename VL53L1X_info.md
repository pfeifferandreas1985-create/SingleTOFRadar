# VL53L1X Sensor Information

Based on the code in `lidar.ino`, the sensor used is the **Pololu VL53L1X Time-of-Flight Distance Sensor Carrier**.

## Dimensions (Pololu Carrier Board)
- **Size:** 13 mm × 18 mm × 2 mm (0.5″ × 0.7″ × 0.085″)
- **Weight:** 0.5 g (without header pins)
